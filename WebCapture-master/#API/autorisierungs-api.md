# Autorisierungs-API

_Captured: 2017-12-27 at 13:33 from [www.gruenderszene.de](https://www.gruenderszene.de/lexikon/begriffe/autorisierungs-api)_

Eine Autorisierungs-API ist eine Form von [Application-Programming-Interface (API)](https://www.gruenderszene.de/lexikon/begriffe/application-programming-interface-api): Als Sondertyp der Web-API, ist die Autorisierungs-API eine Schnittstelle zur Autorisierung (Zugriffsrechtsgewahrung) von Benutzern.

Im Bereich der Autorisierung hat sich der so genannte OAuth-Standard etabliert, uber welchen der Benutzer festlegen kann, ob seine Daten Dritten uber APIs zuganglich sind (zum Beispiel ob eine externe Anwendung [Tweet](https://www.gruenderszene.de/lexikon/begriffe/tweet) in seinem Namen posten darf).

Beispiel

Bekannte Beispiele fur Autorisierungs-APIs sind zum Beispiel Facebook Connect, Google FriendConnect oder der OpenID-Standard, mit denen man sich den Aufbau eines eigenen User-Pools spart, indem sich die Nutzer uber andere Plattformen einloggen.
